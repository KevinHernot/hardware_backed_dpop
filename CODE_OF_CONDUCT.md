# Code of Conduct

## Our Standard

We want `hardware_backed_dpop` to be a respectful, welcoming project for contributors of all backgrounds and experience levels.

## Expected Behavior

- be respectful and constructive
- assume good intent
- give actionable feedback
- help keep the project understandable

## Unacceptable Behavior

- harassment or discrimination
- insulting or demeaning language
- deliberate disruption of community spaces

## Enforcement

Project maintainers may remove content or contributions that violate this code of conduct.
